# @geoalgeria/banques

**كل بنك في الجزائر — كبيانات قابلة للتثبيت.**

[![npm](https://img.shields.io/npm/v/@geoalgeria/banques)](https://www.npmjs.com/package/@geoalgeria/banques)
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

</div>

**21 بنكا** و**8 مؤسسات مالية** معتمدة من بنك الجزائر — مع الأسماء بالفرنسية
والعربية، **`bank_code`** المكوّن من 3 أرقام (RIB)، نوع الملكية (عمومي / أجنبي /
خاص وطني) + **المجموعة المسيطرة**، **SWIFT/BIC**، الموقع الإلكتروني، عنوان
المقر الرئيسي، والربط بالولاية. مُجمَّع من القائمة الرسمية لبنك الجزائر
(الجريدة الرسمية رقم 9، 6 فبراير 2026) ومن الموقع الخاص بكل مؤسسة. جزء من
[GeoAlgeria](https://github.com/yasserstudio/geoalgeria).

```bash
npm install @geoalgeria/banques
```

```js
import banques from "@geoalgeria/banques";

banques.banks();              // 21 بنكا معتمدا
banques.institutions();       // 8 مؤسسات مالية
banques.all();                // 29، البنوك أولا
banques.byId("BNA");          // → Banque Nationale d'Algérie (بالمعرّف أو الاختصار)
```

## المحتوى

| مجموعة البيانات | العدد | ملاحظات |
| --- | --- | --- |
| البنوك | **21** | 7 عمومية · 14 بملكية أجنبية — رمز البنك RIB، الاسم FR/AR، الملكية + المجموعة، البلد، SWIFT/BIC، المقر |
| المؤسسات المالية | **8** | إيجار تمويلي، إعادة تمويل وقرض تعاضدي (بدون ودائع) |
| الوكالات | **1 704** | **كل البنوك الـ 21** — الاسم، العنوان، الهاتف، الولاية، الإحداثيات؛ 1 213 مُرمَّزة جغرافيا؛ **67/69 ولاية** |

يحمل كل سجل `wilaya_code` (المقر الرئيسي) مرتبطا بنموذج 69 ولاية في
[`geoalgeria`](https://www.npmjs.com/package/geoalgeria).

## المصدر والشفافية

- **السجل** هو القائمة المعتمدة الحالية لبنك الجزائر، تم التحقق منها عند
  21 بنكا + 8 مؤسسات (Ziraat Bankası، المعتمد في يناير 2025، هو أحدث بنك).
- الحقول التي لم يمكن تأكيدها من مصدر رسمي هي **`null`**، ولا يتم تخمينها
  أبدا — معظم البنوك الأجنبية لا تنشر اسما عربيا؛ المؤسسات المالية لا تحمل
  SWIFT/BIC؛ بعض عناوين المقرات تقريبية وقد يتم تحسينها.
- قيم SWIFT/BIC هي متغيرات المقر الرئيسي (`…XXX`).
- **إحداثيات الفروع** تُحفظ فقط عندما تقع داخل الجزائر (وبالنسبة للبنوك التي
  يكتفي محدد مواقعها بالترميز الجغرافي للعناوين، تتوافق مع الولاية المُصرَّح
  بها للفرع)؛ وإلا تُحذف النقطة وتُحفظ الولاية — لا إحداثيات مُخمَّنة أبدا.
  يتم جلب صفحات المحدد مع تعطيل التحقق من TLS (`curl -k`) لأن عدة خوادم
  بنكية `.dz` تقدم شهادات معطوبة.
- **بنوك بدون إحداثيات** (BNH، HBTF، Fransabank، BEA، SGA): تُشحن وكالاتها مع
  `lat`/`lng` `null` وولاية مستخرجة من المنطقة في نهاية العنوان. محدد مواقع
  **AGB** محمي بتحدي مضاد للروبوتات، لذا تُلتقط وكالاته الـ 63 عبر متصفح
  headless وتُحدَّث يدويا؛ **Arab Bank** ينشر نقاطا على مستوى المدينة فقط
  (الاسم + الإحداثيات، بدون عنوان). **BDL** و**Trust Bank** مصدرهما خرائط
  Google My Maps (KML) المنشورة من كل بنك؛ **Citibank** و**HSBC** و**Ziraat**
  هي مكاتبهم الوحيدة في الجزائر العاصمة.
- **`bank_code`** هو رمز البنك RIB المكوّن من 3 أرقام (المواضع 5–7 من IBAN)،
  تم التحقق منه مقابل جداول رموز بنكية مستقلة — لا يوجد سجل رسمي عمومي واحد.
  BNH وZiraat (كلاهما حديث الاعتماد) ليس لهما رمز منشور بعد → `null`. المؤسسات
  المالية ليست مؤسسات إيداع، لذا لا تحمل `bank_code`.

## خارطة الطريق (هذا الحزمة)

السجل هو الطبقة الأولى؛ **الوكالات** تغطي الآن **كل البنوك الـ 21 /
1 704 وكالة** (67/69 ولاية). كل بنك ينشر شبكته عبر محدد مواقع مختلف —
Joomla `com_mymaplocations`، WordPress store-locator / إضافات خرائط، حزمة
Vite SPA، صفحات ASP.NET/TYPO3، JSON مضمّن، Google My Maps KML — لذا تُضاف
البنوك مستخرِجا تلو الآخر (انظر `scripts/fetch.mjs`). المحددات بدون إحداثيات
تُشحن مع `lat`/`lng` `null` (الولاية مُستنتجة من العنوان)؛ عندما لا تتوافق
إحداثيات المصدر مع الولاية المُصرَّح بها للفرع، تُحذف الإحداثيات بدلا من
شحنها خاطئة. لا يزال قادما تحت نفس `@geoalgeria/banques`:

- **الصرافات الآلية (DAB/GAB)** — حيث تنشر البنوك محددات مواقع؛ مع شفافية
  تامة حول الاكتمال، إذ لا يوجد في الجزائر دليل عمومي موحد للصرافات الآلية.

فقط. البيانات مُجمَّعة من مصادر عمومية ومقدمة **«كما هي»، بدون ضمان** — وقد
تكون ناقصة أو قديمة. **تحقق من `bank_code` / SWIFT-BIC وتفاصيل الفروع من
المصدر الرسمي قبل أي استخدام مالي أو دفع أو KYC أو امتثال.** لا شيء هنا
